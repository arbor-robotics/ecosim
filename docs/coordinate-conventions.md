Coordinate conventions in Unity are different than in ROS. We will strive to label which convention we are following within code and documentation.

## Unity conventions
- +x to the right
- +y up
- +z forward

{: .important }
Unity's coordinates are left handed.

## ROS conventions
- +x forward
- +y to the left
- +z up
